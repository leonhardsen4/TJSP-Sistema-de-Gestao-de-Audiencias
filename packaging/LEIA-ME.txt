SISTEMA DE GESTÃO DE AUDIÊNCIAS - TJSP (Comarca de Cotia)
Distribuição autocontida — já vem com o Java embutido.

=== O QUE É ===
Este pacote traz tudo o que precisa: o sistema e um Java próprio (portátil).
NÃO é necessário instalar Java na máquina, nem ter direitos de administrador.

=== COMO INICIAR ===
Windows: dê dois cliques em "Iniciar-Sistema.bat"
Linux:   execute "./iniciar-sistema.sh" em um terminal

Aguarde alguns segundos: o navegador abre sozinho em http://localhost:8080
Deixe a janela/terminal ABERTO enquanto usar o sistema.
Para encerrar: feche a janela (Windows) ou pressione Ctrl+C (Linux).

=== DADOS E BACKUP ===
Os dados ficam na pasta "data" (arquivo tjsp_audiencias.db), criada ao lado
do programa. Os backups vão para a pasta "backups". Para levar o sistema
para outra máquina COM os seus dados, copie a pasta inteira.

=== ACESSO POR OUTROS COMPUTADORES (rede local) ===
Para outras pessoas acessarem, elas abrem o navegador em
http://IP-DESTA-MAQUINA:8080  (ex.: http://192.168.0.10:8080).
Pode ser necessário liberar a porta 8080 no Firewall do Windows, e a rede
precisa permitir a conexão entre as máquinas (ver observações abaixo).

=== OBSERVAÇÕES ===
- Porta padrão: 8080. Para mudar, defina a variável de ambiente PORT.
- Requer Windows 64 bits ou Linux 64 bits (o Java embutido é para x64).
- Em máquinas muito restritas (antivírus/AppLocker), a execução de programas
  não homologados pode ser bloqueada — isso é política da instituição, não
  uma limitação do sistema.
